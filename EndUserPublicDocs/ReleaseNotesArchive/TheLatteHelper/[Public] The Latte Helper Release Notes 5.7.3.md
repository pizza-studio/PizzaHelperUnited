// 《拿铁小助手》v5.7.3 的更新内容简述：

- 开箱画面（OOBE）：紧急修复了该画面在 iPhone SE3 开启放大模式时排版错误暨翻页按钮无法正常显示的故障。
- 抽卡记录管理器：紧急修复了 iOS 设备在获取抽卡记录时因为设备画面自动锁定而触发的可能的资料毁损故障。从这一版更新开始，在获取抽卡记录的过程中，本软件会试图让整个屏幕都亮着。
- UI: 对 iPhone 专用的屏幕底部页签条在 Liquid Glass 模式下的表现做了进一步的美术改进。
- UI: 解决了在 macOS 系统下鼠标对部分页面内容的拖拽行为会过于高频地触发页面重载响应动作的故障。
- UI: 调整了展柜陈列画面的资料加载进度条的镶嵌方式。
- 官方游戏活动清单页面现允许用户藉由 App 设定画面强制清理与相关资讯有关的本机缓存。这些缓存内容的存储位置也从 UserDefaults 挪至文件系统，减轻 UserDefaults 压力。
- 官方游戏活动清单页面右上角新增了选单命令、允许用户过滤掉那些没在本地账号登记内容当中牵涉到的游戏的内容。打比方说你当前设备登记的本地帐号只有 A 与 B 的话，则该功能会让官方游戏活动清单页面排除掉与这些游戏无关的官方游戏活动。
- 官方游戏活动清单页面现对官方游戏活动清单页面做了排版修整。部分活动会追加标题文字显示、以应对其缩略图缺乏文字的情况。
- 修复了 Enka 展柜在翻阅角色面板时因整个容器视图重建而打断翻页动画的故障。
- 玩家角色库存页面补回了一个功能「导出角色库存报表图」：允许用户将当前页面显示的所有角色清单都导出成一张图片。出于一些技术上的顾虑，该功能自 v3.0.1 版被移除、且在 v5.0.0 重写整个 App 时没被实作。现在有了应对这些顾虑的技术手段，所以就把这个功能重新实作了。
- 玩家角色库存页面在以展开模式显示时，现在会在按照角色元素属性来分类、且显示这些元素的名称。
- 玩家角色库存页面从角色肖像点击进入米游社 / HoYoLAB 角色面板时的导航行为现调整为与 Enka 展柜相同的行为。
- 玩家角色库存页面的金数统计不再包括奇偶。

(现阶段暂停提供 macCatalyst 版本，以减轻 App Store 审委会的审核工作量。这导致该 App 目前无法支持 Intel Mac 机种，因为只有 Apple Silicon Mac 可以直接运行 iPad 应用。上文中讨论到的与 macOS 有关的内容更新均指该 App 的 iPadOS 版本在 macOS 系统下的行为。)

// Major changes introduced in The Latte Helper v5.7.3:

- OOBE (Out-of-Box Experience): Urgently fixed layout issues on devices with limited screen canvas (such as iPhone SE3 with magnification enabled), where page buttons were not properly visible.
- Gacha Record Manager: Urgently fixed a potential data corruption issue that occurred when the device screen automatically locked during gacha record retrieval. Starting from this version, the app will attempt to keep the screen on throughout the gacha retrieval process.
- UI: Further improved the Liquid Glass design of the iPhone-specific global tab bar.
- UI: Fixed an issue on macOS where mouse dragging on certain pages triggered excessive page reload actions.
- UI: Adjusted the layout of the loading progress indicator in the Showcase list view section.
- Official Game Events List: Related local cache can now be force-cleaned through App Settings. Cache data has also been migrated from UserDefaults to the file system, reducing the burden on UserDefaults.
- Official Game Events List: Added a new menu command in the top-trailing corner, allowing users to filter events based on registered local profiles. For example, if your device only has profiles for games A and B, events for other games will be excluded.
- Official Game Events List: Refined the layout of the events list. Some events now display additional title text to compensate for thumbnails that lack readable text.
- Fixed an issue where Enka-based Character Showcase animations were disrupted when flipping between character build pages.
- Character Inventory: Restored the character inventory export feature that allows you to save the entire character list as a single image (Character Inventory Report). This feature was removed in v3.0.1 due to technical concerns and was not re-implemented during the v5.0.0 redevelopment. Technical solutions have now been implemented, enabling the re-introduction of this feature.
- Character Inventory: In expanded view mode, characters are now grouped by their elemental attributes with element names displayed.
- Character Inventory: Navigating to HoYoLAB-Miyoushe Character Showcase from character portraits now opens in the same navigation context instead of as a popup.
- Character Inventory: Gold statistics now exclude Wonderland Manekins from five-star character ownership counts.

(We stopped supplying the macCatalyst build at App Store to reduce the App Review workload. As a result, the app is currently unavailable on Intel-based Macs. Only Apple Silicon Macs are capable of running the iPad app. All macOS-related mentions above refer to the behavior of the iPadOS version running on macOS.)

// 「ラテ助手」v5.7.3 の主な更新内容：

- ようこそ画面（OOBE）：iPhone SE3 などの画面倍率が制限される端末において、レイアウトエラーが発生し、ページボタンが表示されない問題を緊急修正しました。

- UI：iPhone専用のグローバルタブバーの Liquid Glass デザインをさらに改善しました。
- UI：macOS でマウスドラッグが一部ページで過度に高い頻度でページリロード動作を引き起こす問題を修正しました。
- UI：ショーケースリストビューのロード進度インジケーターのレイアウトを調整しました。
- 公式ゲームイベントリスト：関連するローカルキャッシュを App 設定から清除できるようになりました。キャッシュデータも UserDefaults からファイルシステムに移行され、UserDefaults の負荷が軽減されました。
- 公式ゲームイベントリスト：ページ右上のメニューコマンドで、登録されたローカルプロフィールに関連するゲームのイベントのみをフィルタリングできるようになりました。例えば、デバイスに A と B のプロフィールのみが登録されている場合、他のゲームのイベントが除外されます。
- 公式ゲームイベントリスト：イベントリストのレイアウトを改善しました。テキストが不足しているサムネイル用に追加のタイトルテキストが表示されるようになりました。

- キャラクター一覧：その画面の全部の内容を 1 つの画像としてエクスポートする機能「キャラクター一覧レポート画像のエクスポート」を再実作しました。この機能は v3.0.1 で削除され、v5.0.0 の再開発時に実装されていません。技術的な解決策が実装されたため、この機能を作り直しました。
- キャラクター一覧：展開表示モードでは、キャラクターが元素属性でグループ化され、要素名が表示されるようになりました。
- キャラクター一覧：キャラクターのアバターから HoYoLAB-米遊社 ショーケースへナビゲートする際、ポップアップではなく同じナビゲーションコンテキストで開くようになりました。
- キャラクター一覧：金数の統計は幻境ドールを除外するようになりました。

（App Review の作業負荷を軽減するため、macCatalyst 版の提供を一時停止しています。そのため現時点では Intel 製 Mac ではご利用できませんので、ご了承くださいませ。Apple Silicon 機種では iPad アプリを使うのは可能です。上記の macOS に関する全ての説明は、macOS 上で実行されている当アプリの iPadOS 版の動作に関連しています。）

// Основные изменения в «Латте помощник» v5.7.3:

- OOBE (первоначальная настройка): Экстренно исправлена ошибка макета экрана при ограниченном размере экрана (например, iPhone SE3 с включенным увеличением), когда кнопки переключения страниц были невидимы.
- Менеджер истории получения: Экстренно исправлена потенциальная ошибка повреждения данных, возникающая при автоматической блокировке экрана во время получения истории получения. Начиная с этой версии, приложение будет стараться держать экран включенным на протяжении всего процесса получения истории.
- Интерфейс: Дальнейшее улучшение дизайна Liquid Glass глобальной вкладки для iPhone.
- Интерфейс: Исправлена ошибка на macOS, при которой перетаскивание мышью на некоторых страницах вызывало чрезмерно частые действия перезагрузки страницы.
- Интерфейс: Отрегулирован макет индикатора хода загрузки в разделе списка витрин.
- Список официальных игровых событий: Связанный локальный кэш теперь можно очищать принудительно через параметры приложения. Данные кэша также были переведены из UserDefaults в файловую систему, что снижает нагрузку на UserDefaults.
- Список официальных игровых событий: Добавлена новая команда меню в правый верхний угол, позволяющая пользователям фильтровать события на основе зарегистрированных локальных профилей. Например, если на вашем устройстве зарегистрированы профили только для игр A и B, события для других игр будут исключены.
- Список официальных игровых событий: Улучшен макет списка событий. Некоторые события теперь отображают дополнительный текст заголовка для компенсации отсутствия текста в миниатюрах.
- Исправлена ошибка, при которой анимация перевораживания страниц в витрине персонажей Enka была нарушена при переключении между страницами сборок персонажей.
- Инвентарь персонажей: Восстановлена функция экспорта всего списка персонажей в виде одного изображения (отчет об инвентаре персонажей). Эта функция была удалена в v3.0.1 и не была переиспользована при переразработке v5.0.0. Технические решения были реализованы, позволяющие восстановить эту функцию.
- Инвентарь персонажей: В режиме развернутого представления персонажи теперь группируются по стихийным атрибутам с отображением названий элементов.
- Инвентарь персонажей: При навигации к витрине персонажей HoYoLAB-Miyoushe из портретов персонажей теперь открывается в том же контексте навигации, а не во всплывающем окне.
- Инвентарь персонажей: Статистика золота теперь исключает Манекенов из подсчета владельцев пятизвездочных персонажей.

(Временно приостановлен выпуск версии macCatalyst, чтобы снизить нагрузку на команду App Review. Поэтому приложение сейчас недоступно на компьютерах Mac с процессорами Intel. Apple Silicon Mac могут запускать приложение iPad напрямую. Все упоминания macOS выше относятся к поведению версии для iPad этого приложения, работающей на macOS.)
